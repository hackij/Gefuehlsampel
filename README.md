# Gefühlsampel

Eine einfache Web-App für den Berufsschulunterricht. Schülerinnen und Schüler wählen eine von fünf Stimmungen aus und bekommen das passende Emoji groß auf dem Display angezeigt.

## Nutzung

Öffne `index.html` direkt im Browser oder lade den gesamten Ordner auf einen Webserver hoch. Die App benötigt kein Konto, keine Datenbank und keine Internetverbindung.

## Stimmungen

- Fröhlich
- Müde
- Gelangweilt
- Gestresst
- Geht so
